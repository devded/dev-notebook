# Java Tutorials

Long-form, example-driven tutorials with diagrams. Each one teaches a topic from the
ground up — mental model, runnable Java 17/21 code, mermaid diagrams, common pitfalls,
and best practices — and pairs with the matching [interview questions](../questions/)
set for drilling afterward.

## Language core

- [Basics & Syntax](./basics) — JVM/JRE/JDK, types, `var`, value vs reference, `==` vs `equals`
- [Core Language & Modern Java](./core-language) — records, sealed types, pattern matching, switch expressions, text blocks
- [Object-Oriented Programming & Design](./oop-design) — the four pillars, interfaces, SOLID
- [Generics](./generics) — bounded types, wildcards, PECS, type erasure
- [Exceptions & Error Handling](./exceptions) — checked vs unchecked, try-with-resources, custom exceptions
- [Strings & Text](./strings) — immutability, String pool, StringBuilder, regex, encoding

## Data & functional

- [Collections Framework](./collections) — List/Set/Map internals, HashMap, choosing the right one
- [Streams & Functional Programming](./streams-functional) — lambdas, Stream pipeline, Collectors, Optional

## Runtime & platform

- [Concurrency & Multithreading](./concurrency) — JMM, executors, locks, virtual threads
- [Memory, JVM Internals & GC](./memory-jvm) — heap/stack, class loading, garbage collectors, JIT
- [I/O, Files & NIO.2](./io-nio) — streams, `java.nio.file`, serialization, channels
- [Date & Time API](./datetime) — `java.time`, zones, Duration/Period, formatting

## Advanced

- [Reflection & Annotations](./reflection-annotations) — runtime inspection, dynamic proxies, custom annotations
- [Design Patterns in Java](./design-patterns) — GoF patterns, modernized with lambdas/records/sealed
