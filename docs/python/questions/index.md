## Format for adding questions

Each question is a heading (always visible) followed by its answer. Paste new ones using this template:

````md
## Your question here? <Badge type="tip" text="easy" />

Your concise, paraphrased answer. Inline `code` is fine.

```python
# optional code block
```
````

Badge types: `tip` (easy), `warning` (medium), `danger` (hard).
